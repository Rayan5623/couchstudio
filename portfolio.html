The Studio

Built from the couch

Couch Studio is a one-person indie studio based in Rome, Italy. Apps, mobile games, Roblox experiences — built with curiosity and shipped fast.

The philosophy is simple: launch MVPs, learn from real users, iterate. No bloat, no committee decisions. Just ideas turned into products.

Flutter React Godot Roblox / Lua Node.js Next.js
🎯 Philosophy

Ship early, ship often. A working MVP beats a perfect product that never launches.

🛠️ Stack

Flutter for mobile, React for web, Godot for games, Lua for Roblox. Always the right tool for the job.
